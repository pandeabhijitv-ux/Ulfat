## üõ†Ô∏è How to Capture Screenshots:

### Method 1: Browser Developer Tools (Best Quality)
1. Open http://localhost:8000 in Chrome
2. Press F12 to open DevTools
3. Click device toolbar icon (or Ctrl+Shift+M)
4. Set device to "Pixel 5" or custom 1080x1920
5. Navigate through app and capture screens
6. Use Windows Snipping Tool or browser's screenshot feature

### Method 2: Phone Screenshot
1. Deploy app to your Android phone
2. Take screenshots directly on phone (Power + Volume Down)
3. Screenshots will be in native resolution

### Method 3: Android Emulator
1. Open Android Studio Emulator
2. Install the APK
3. Use emulator's screenshot button
4. Save as PNG
